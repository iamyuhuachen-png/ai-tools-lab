## 📁 工具列表

### 1. 👟 鞋面變體生成器 (Shoe Variant Generator)
- **路徑**: `/shoe-variant-generator/`
- **用途**: 快速探索鞋類設計的各種變體可能性
- **功能**:
  - 選擇要變動的鞋面部位（鞋帶、鞋頭、鞋舌、鞋口等）
  - 從預設選項選擇或輸入自訂樣式
  - 自動生成精確的提示詞
  - 搭配 Nano Banana Pro (Gemini 3 Pro Image) 使用
- **建立日期**: 2026-01-02

## 🚀 使用方式

2. 點選想要使用的工具
3. 依照工具介面操作
